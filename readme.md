# ML Group Project: Player Performance Prediction

Predicting Expected Goals (xG) using Advanced Machine Learning Models.

## Project Overview

This project focuses on predicting **Expected Goals (xG)** for football players based on performance statistics from the 2025-2026 season. By leveraging various machine learning architectures, we aim to quantify the quality of scoring opportunities and player efficiency.
